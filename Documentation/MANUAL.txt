Quick setup
===========

Sink is installed pretty easily using the standard Python way:

>	python setup.py install

this will install the `sink.main` and `sink.tracking` modules in your Python
library and make the `sink` command available.

Uses
====

Sink is primarily a tool to *detect changes multiple between directories*
